Version 0.0.1

Version 0.0.1 was released on 15/09/2016. Its the first stable release with all tools and services.

Changes in This Release

Current Features

  • Support for multiple clients.

  • Support for client internet connections.

  • Support dedicated configuration per client.

  • Support jobs for a client and global jobs.

  • Support to Incremental Backup, Full Backup and Restore

  • Support to Volume Shadow Service

  • Support to Custom Modules

  • Support to multiple repositories

  • Support to multiple images per repository

  • Support to custom jobs for the client, server or storage

  • Flat File System for the storage, to store the backups.

  • Auto-backup to the services catalog

  • DrawerBackup Manager Web Interface, to manage multiple storages and servers.

  • CLI Tools to manage the server, storage and to configure the client

  • Support to multiple database engines and to custom database engines

  • DrawerBackup Manager Web Interface, to manage multiple storages and servers.

  • Fast backup and fast restore with advanced filters and options.
